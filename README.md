Scripts of paper "Changes of gut microbiota reflect the severity of major depressive disorder: a cross sectional study".

  - numpy 1.17.3
2. 2_PCoA.R
  - R >= 4.1.2
  - vegan
  - ggplot2
  - ggrepel

  - R >= 4.1.2
  - microbiome
  - DirichletMultinomial
  - reshape2
  - magrittr
  - dplyr
  - readxl
  - Cairo
